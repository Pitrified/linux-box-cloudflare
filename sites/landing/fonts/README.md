# Self-hosted fonts

IBM Plex Mono (400, 500) and IBM Plex Sans (variable, 300-400), latin subset,
as served by Google Fonts (woff2 builds of the upstream IBM Plex release).

Copyright IBM Corp., licensed under the SIL Open Font License 1.1 - see
[`OFL.txt`](OFL.txt) (the license text from the upstream repo,
https://github.com/IBM/plex). The OFL permits self-hosting and redistribution
of the font files, including in a public repo like this one, provided the
copyright notice and license accompany them - which is this folder's job.
The fonts may not be sold on their own; bundling with the site is fine.
